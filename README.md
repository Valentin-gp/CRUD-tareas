# CRUD de Tareas en PHP + MySQL

Proyecto simple de CRUD para gestionar tareas.

## Requisitos

- PHP 7.4 o superior
- MySQL o MariaDB
- Servidor web (Apache recomendado)

## Instalaci√≥n

1. Clonar el repositorio:
git clone https://github.com/Valentin-gp/CRUD-tareas


2. Copiar `config.example.php` como `config.php`.

3. Editar `config.php` con tus credenciales reales del servidor.

4. Importar `tareas_db.sql` en tu base de datos.

5. Subir los archivos a tu hosting.
